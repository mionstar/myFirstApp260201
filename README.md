# myFirstApp260201

Claude Code の first step として作成した習作プロジェクトです。

現在時刻と曜日を表示するWebページ。

## 機能

### 基本表示
- 「hello」に続いて現在の曜日と時刻を表示
- 1秒ごとに時刻を更新

### 曜日による背景色
- 土曜日: 青色
- 日曜日: 朱色
- 平日: グレー

### 夜間演出（18:00〜6:00）
- 背景が深い紺色の夜空に変化
- 黄色とピンクの星が瞬くアニメーション（ピンクは15%の出現率）
- 文字色が白に変化
- 夜間演出は曜日の背景色より優先

## 使い方

`index.html` をブラウザで開く
